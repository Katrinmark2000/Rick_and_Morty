# Elfsight Test Task (Rick and Morty API)
Интерактивное веб-приложение на React, которое позволяет искать и фильтровать персонажей из Rick and Morty API

Можно посмотреть здесь https://katrinmark2000.github.io/Rick_and_Morty/

## Функциональность

- Фильтрация персонажей по нескольким параметрам
- Поиск по имени и типу
- Пагинация
- Просмотр детальной информации о персонаже
- Сохранение состояния фильтров в URL
- Адаптивный дизайн

## Технологии

- React 18
- JavaScript
- Styled-components
- Rick and Morty API

## Установка и запуск

1. Клонировать репозиторий:

```bash
git clone https://github.com/Katrinmark2000/Rick_and_Morty
```

2. Установить зависимости:

```bash
npm install
```

3. Запустить приложение:

```bash
npm run start
```
